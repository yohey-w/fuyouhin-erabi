## 「複数社に見積もりを取ったら3倍の差があった」

X（旧Twitter）で「不用品回収を複数社に見積もりを取ったら3倍の差があった」という声が多く見られる。同じ量の荷物でも、業者によって30,000円から90,000円まで差が出ることがある。

45歳の小林さん（仮名）は不用品回収の見積もりが高かった。値段交渉できるのか、どう交渉すればいいか知りたいと思っている。

このガイドでは、不用品回収の費用を下げるための「交渉術」と「正しい方法」を解説する。

## 最強の交渉術は「相見積もり」

まず大前提として、**値引き交渉の最も効果的な方法は相見積もり**だ。

3社から見積もりを取ってから、「他社では○○円でした。同じ金額か安くできますか？」と伝えるだけで、多くの業者は価格を下げてくれる。

### 相見積もりの実践方法

1. **3社から同じ条件で見積もりを取る**：品目・量・作業内容を統一して比較
2. **書面（メール・LINEでも可）で見積もりを受け取る**：後の交渉で使える証拠になる
3. **一番安い金額を基準に交渉する**：「A社がこの金額でした。合わせてもらえますか？」

## 値下げ交渉が通りやすい状況

### 状況1：繁忙期以外（10月〜12月、3月は繁忙期）

引っ越しシーズン（3月）や大掃除シーズン（12月）は業者が忙しく、値引きに応じにくい。逆に**4〜7月・9〜10月**は比較的空いており、値引き交渉が通りやすい。

### 状況2：まとめて大量に依頼する

少量よりも多量の方が、業者にとっての効率が良い。「他にもこれだけあるが、全部まとめてやってもらえるなら○○円にしてもらえるか？」という交渉は通りやすい。

### 状況3：平日の午前中を指定できる場合

週末は予約が集中する業者が多い。平日の午前中は比較的空きがある場合が多く、「平日でよければ安くなりますか？」という交渉が有効な場合がある。

### 状況4：急ぎでない場合

「来週でも再来週でも構いません」という柔軟な姿勢を示すと、業者の都合が良いタイミングで入ってもらえるため、割引を提案してくれることがある。

## 交渉のNGパターン：やってはいけないこと

### NG1：根拠のない値切り

「もっと安くして」だけでは業者も対応できない。**相見積もりの金額や具体的な理由を提示することが前提**。

### NG2：作業当日の値切り

作業が始まってから「もっと安くして」と言うのは最も迷惑。業者の信頼を失い、最悪の場合、作業を途中で止められることもある。

### NG3：強引な態度や脅しのような交渉

「他に頼む」「口コミに書く」などの脅しめいた発言は逆効果。業者が断る可能性が高まるだけでなく、トラブルに発展することもある。

### NG4：口頭だけの合意で進める

「○○円でいいと言ってくれた」という口頭の合意は後で言った言わないになりやすい。必ず書面（見積書）で確認する。

## 費用を下げるための5つの方法

### 方法1：相見積もり（最強）

前述の通り、相見積もりが最も効果的。

### 方法2：品目を減らす

回収する品目が少ないほど費用は下がる。自分で処分できるものは先に処分しておく。
- 燃えるゴミに出せる小物は自治体のゴミ回収で処分
- 動作品の家電はリサイクルショップに売る
- 粗大ゴミに出せるものは自治体回収を活用

### 方法3：繁忙期を避ける

3月・12月を避けるだけで費用が変わることがある。

### 方法4：分解・梱包で作業効率を上げる

大型家具を事前に分解しておくと作業時間が短くなり、費用が下がる業者もある。

### 方法5：割引キャンペーンを活用する

業者によっては定期的に割引キャンペーンを実施している。メルマガ登録や公式サイトのチェックで情報をつかむ。

## 見積もり比較表の作り方

複数社を比較する際は、以下の項目を揃えて比較する。

| 比較項目 | A社 | B社 | C社 |
|---------|-----|-----|-----|
| 基本料金 | | | |
| 品目別追加料金 | | | |
| 作業費 | | | |
| 廃棄・処分費 | | | |
| 交通費・搬出費 | | | |
| **合計（税込）** | | | |

**「合計金額」で比較する。** 基本料金が安くても追加費用が多い業者には注意。

## まとめ：不用品回収の費用を下げる最短ルート

1. **3社から相見積もりを取る**（書面で）
2. **最安値を基準に他社に交渉する**
3. **品目を事前に減らしておく**
4. **繁忙期を避けて平日を指定する**

「もっと安くして」だけの交渉は通らない。根拠のある比較と、業者に「やりたい」と思ってもらえる条件を提示することが交渉の本質だ。
