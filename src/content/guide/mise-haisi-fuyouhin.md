## 業務用品の産廃/一廃区分表

**基本ルール：** 事業活動で発生した廃棄物は原則として産業廃棄物。ただし例外があります。

| 廃棄物の種類 | 区分 | 処理方法 |
|---|---|---|
| 厨房機器（冷蔵庫・冷凍庫・食洗機等） | 産業廃棄物（廃電気機械器具） | 産廃業者 or 買取 |
| フライヤー・調理機器（電気系） | 産業廃棄物 | 産廃業者 or 買取 |
| ガス機器（コンロ・オーブン等） | 産業廃棄物（廃金属等） | 産廃業者 |
| テーブル・椅子（木製・スチール） | 一般廃棄物 | 不用品回収業者 or 買取 |
| 食器・調理器具（家庭用相当サイズ） | 一般廃棄物 | 不用品回収 or 廃棄 |
| 看板・POP用品 | 産業廃棄物（廃プラスチック等） | 産廃業者 |
| 食用油の廃油 | 産業廃棄物（廃油） | 廃油専門業者 |
| 食材の廃棄（生ごみ等） | 産業廃棄物（事業系一般廃棄物）※ | 一般廃棄物処理業者 |
| 段ボール・紙類 | 産業廃棄物（廃紙）or リサイクル | 資源回収業者 |
| 厨房用消火器 | 産業廃棄物（廃危険物） | 消火器専門業者 |

※飲食店の食品廃棄物は「事業系一般廃棄物」として自治体の許可業者に依頼

**まとめると：**
- テーブル・椅子など家具什器 → 一般廃棄物（不用品回収業者でOK）
- 厨房機器・電気機械・ガス機器 → 産業廃棄物（産廃業者または買取）
- 廃油・廃液 → 産業廃棄物（専門業者必須）

## 厨房機器の買取可否の目安

廃棄前に買取査定を依頼することで費用を相殺できる可能性があります。

### 買取可能性が高いもの

| 品目 | 買取可能性 | 条件 |
|---|---|---|
| 業務用冷蔵庫・冷凍庫 | 高 | 動作品・製造10年以内 |
| 業務用製氷機 | 高 | 動作品 |
| エスプレッソマシン | 高 | ブランド品 |
| 業務用食洗機 | 中 | 動作品・清掃状態良好 |
| フライヤー | 中 | 動作品・清掃状態良好 |
| ショーケース・陳列棚 | 中 | 美観状態による |
| テーブル・椅子（状態良好） | 低〜中 | セット・デザイン性あるものは売れやすい |

### 買取しにくいもの

- 製造15年以上の機器
- 故障品・汚れが激しいもの
- 特定の店舗仕様（オーダーメイド什器等）
- 家庭用品として需要がないもの

**対応する買取業者：**
- 飲食店専門の業務用厨房機器買取業者
- 中古厨房機器販売店
- リサイクルショップ（一部）

## 閉店片付けの段階別ステップ

### Step 1：棚卸し（閉店2〜3ヶ月前）

全設備・備品をリストアップします：
- 品目・状態・製造年を記録
- 産廃/一廃の仮区分
- 買取対象候補品の選別

### Step 2：買取査定依頼（閉店1〜2ヶ月前）

- 業務用厨房機器買取業者に連絡
- 状態確認・見積もり（出張査定が多い）
- 買取品目が確定したら日程調整

### Step 3：産廃処理の手配（閉店1ヶ月前）

- 産業廃棄物収集運搬業者に連絡・見積もり
- 廃油・廃液がある場合は専門業者の別途手配
- 消火器の回収手配（近くのホームセンターやメーカー窓口）

### Step 4：一廃（家具什器等）の回収（閉店直前〜閉店後）

- 不用品回収業者に連絡・見積もり
- テーブル・椅子・棚等を一括回収
- 閉店後の空き店舗状態での作業も可能

### Step 5：清掃・原状回復（閉店後）

- 業者回収完了後、清掃業者を手配
- テナントの場合は原状回復義務の確認

## 費用の目安

| 内容 | 費用目安 |
|---|---|
| 一般廃棄物（家具・什器）の不用品回収 | 2トン車1台：40,000〜80,000円 |
| 産業廃棄物（厨房機器等）処理 | 機器の量・種類による（10万〜50万円以上） |
| 廃油処理 | 量による（2万〜10万円） |
| ハウスクリーニング（店舗） | 50〜200㎡で10万〜50万円 |

### 買取で費用を相殺

状態の良い業務用冷蔵庫は5万〜20万円で買い取ってもらえることもあります。
買取査定を積極的に活用することで、廃棄費用を大幅に圧縮できます。

## よくある疑問

**Q. 閉店後も厨房機器を残置したままにできますか？**

A. テナント契約では原状回復義務があります。設備を残置したまま退去すると追加費用が発生します。閉店日が確定したら早めに片付けを開始してください。

**Q. フランチャイズ店ですが、設備の処分はフランチャイズ本部に相談すべきですか？**

A. FC契約内容によっては、設備の処分や返却に関してFC本部への事前確認が必要な場合があります。契約書を確認するか、本部に問い合わせてから動いてください。

**Q. 急いで閉店しなければならない事情があります。1週間で片付けられますか？**

A. 可能ですが、各業者の手配を同時並行で進める必要があります。買取業者・産廃業者・一廃業者に同時に連絡し、日程を集中させることで対応できます。

## まとめ

閉店片付けの要点：
- テーブル・椅子等の家具什器 → 一廃（不用品回収業者でOK）
- 厨房機器・電気機械 → 産廃（産廃業者または買取）
- 廃棄前に買取査定を忘れずに（費用圧縮に直結）
- 産廃はマニフェスト対応業者を選ぶ（法的義務）
- 計画は閉店2〜3ヶ月前から始める
